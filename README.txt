1. Run startupscript.sql on oracle server.
2. Change connection info in DatabaseConnection.java if necessary. By default it is set to use the csshrpt server.
3. Run program from Netbeans. The Netbeans project files are present in the RecipeCookBook folder. However, it may be possible that a new project will need to be created or that the ojdbc7.jar will need to be added as a project JAR.
4. After running the program, you will see that sample data already exists in the program. You can use that data however you need to grade the assignment.
